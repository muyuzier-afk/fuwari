# 关于本站

这是一个技术博客，分享 AI、Serverless、云原生等前沿技术的情报、分析与实战教程。

## 技术栈

### 框架 - Astro

本站使用 [Astro](https://astro.build/) 构建，这是一个专注于内容的现代静态站点生成器。

**为什么选择 Astro？**

- **岛屿架构**: 默认零 JavaScript，只在需要交互的组件加载 JS，页面加载速度极快
- **多框架支持**: 可以在同一项目中混用 React、Vue、Svelte 等框架
- **内容集合**: 内置 Markdown/MDX 支持，自动类型安全
- **SEO 友好**: 静态生成，对搜索引擎更友好
- **开发体验**: 热更新快，构建速度快

### 模板 - Fuwari

基于 [Fuwari](https://github.com/saicaca/fuwari) 主题开发，这是一个简洁优雅的 Astro 博客模板。

**主要特性:**

- 响应式设计，适配各种设备
- 深色/浅色模式切换
- 文章归档、标签、分类
- 代码高亮 (Expressive Code)
- 数学公式支持 (KaTeX)
- 平滑页面过渡动画 (Swup)
- 自动生成 Sitemap
- RSS 订阅支持

### 部署平台 - Cloudflare Pages

网站托管在 [Cloudflare Pages](https://pages.cloudflare.com/)，享受全球边缘网络带来的快速访问体验。

**选择 Cloudflare Pages 的理由:**

- **免费额度慷慨**: 无限站点、10w请求/天、无限带宽
- **全球边缘网络**: 300+ 数据中心，就近访问
- **自动 HTTPS**: 免费 SSL 证书，自动续期
- **持续部署**: 连接 GitHub，推送即部署
- **预览部署**: 每个 PR 自动生成预览链接
- **Web Analytics**: 免费的隐私友好型分析

### CDN 加速 - 腾讯云 EdgeOne

使用 [腾讯云 EdgeOne](https://cloud.tencent.com/product/teo) 作为 CDN，确保国内用户也能快速访问。

**EdgeOne 优势:**

- **全球加速**: 2800+ 节点，覆盖全球 70+ 国家和地区
- **智能调度**: 根据用户位置自动选择最优节点
- **DDoS 防护**: T 级别 DDoS 清洗能力
- **WAF 防护**: Web 应用防火墙，防止常见攻击
- **智能缓存**: 自动缓存静态资源，减少源站压力
- **实时监控**: 详细的访问日志和性能分析

### 人机验证 - hCaptcha

集成 [hCaptcha](https://www.hcaptcha.com/) 进行人机验证，保护网站免受机器人和垃圾信息的侵扰。

**为什么选择 hCaptcha？**

- **隐私优先**: 不追踪用户，符合 GDPR
- **免费使用**: 对网站所有者完全免费
- **高准确率**: 有效识别机器人，减少垃圾信息
- **无障碍**: 提供音频验证等无障碍选项
- **全球可用**: 不受地区限制，国内也能正常使用
